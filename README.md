Resource Sentinel

Overview

Resource Sentinel is a Python-based Operating System Monitoring Agent designed to monitor system resources such as CPU, memory, and disk usage in real time.

Problem Statement

Modern computers may become slow or unstable when resource usage becomes too high. Users often notice these issues only after performance has already degraded.

Solution

Resource Sentinel continuously observes system resources, analyzes usage levels, and alerts users when predefined thresholds are exceeded. It also records system activity for future analysis.

Features

Real-time CPU monitoring
Memory usage monitoring
Disk usage monitoring
Alert generation
Log file creation
Performance tracking

Technologies Used

Python
psutil

Agent Cycle

Observation – Collect system resource data.
Thinking – Analyze resource usage and detect abnormal conditions.
Action – Generate alerts and create logs.
